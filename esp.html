Habilidades

HTML5 es un lenguaje de marcado para la web y es una tecnología clave de Internet. Esta nueva versión trae cambios importantes en cuanto al rol del HTML en el mundo web, a través de nuevas funcionalidades como la semántica y la accesibilidad.

CSS3 es la tercera versión de las famosas Cascading Style Sheets, que define estilos para un proyecto web. Con efectos de transición, imagen, imagen de fondo / fondo y otros, puede crear estilos únicos para sus proyectos web, cambiando diferentes aspectos de diseño en el diseño de la página.

JavaScript es un lenguaje de programación estructurado e interpretado. Junto con HTML y CSS, el JavaScript es una de las tres tecnologías principales en la web. JavaScript habilita páginas interactivas y, por lo tanto, es una parte esencial de las aplicaciones web.

React es una biblioteca de JavaScript de código abierto centrada en la creación de interfaces de usuario. (frontend) en páginas web. Es mantenido por Facebook, Instagram, otras empresas y una comunidad de desarrolladores individuales

Git es un sistema de control de versiones distribuido, utilizado principalmente en el desarrollo de software, pero se puede utilizar para registrar el historial de edición de cualquier tipo de archivo.

GitHub es una plataforma de alojamiento de archivos y código fuente con control de versiones mediante Git. Permite a los programadores, utilidades o cualquier usuario registrado en la plataforma contribuir a proyectos privados y / o de código abierto desde cualquier parte del mundo.
